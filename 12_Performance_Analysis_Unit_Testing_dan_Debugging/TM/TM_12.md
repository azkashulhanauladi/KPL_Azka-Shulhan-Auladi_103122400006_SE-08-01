## Tugas
Fungsi di bawah ini melakukan penjumlaha pada penghitung (counter), yang sesederhana menambahk jumlah jika kamu menekan tombol.

`hitung.js`
```
function tambahPengitung(terkini, jumlah) {
  terkini = terkini + jumlah;
  return terkini;
}
```
dan `hitung.test.js`
```
import { test } from 'node:test';
import assert from 'node:assert';
import { tambahPengitung } from './hitung.js';

test('5 tambah 3 sama dengan 8', () => {
  assert.strictEqual(tambahPengitung(5, 3), 8);
});

test('0 tambah 10 sama dengan 10', () => {
  assert.strictEqual(tambahPengitung(0, 10), 10);
});
```

Bisakah kamu tunjukkan apakah kode sudah benar atau bagian mana yang perlu diperbaiki beserta alasannya?

## Deskripsi Program
Awalnya kode ini memang udah lolos test karena hasil return-nya sesuai, tapi sebenarnya ada masalah di cara kerja counternya. Variabel terkini cuma ada di dalam fungsi, jadi setiap kali fungsi dipanggil nilainya dibuat ulang dari awal. Akibatnya, counter nggak benar-benar menyimpan nilai sebelumnya dan nggak punya state yang bertahan antar pemanggilan.

Solusinya adalah menambahkan variabel hitungan di luar fungsi supaya nilainya tetap tersimpan selama program berjalan. Jadi setiap kali fungsi dipanggil, perhitungannya bisa lanjut dari nilai terakhir, bukan mulai dari nol lagi. Selain itu, ditambahkan juga fungsi reset() untuk mengembalikan counter ke nilai awal. Fungsi ini berguna terutama saat testing karena setiap test case bisa dimulai dari kondisi yang bersih dan konsisten.

Parameter fungsi juga dibuat lebih sederhana dengan hanya menerima jumlah sebagai input. Nilai counter saat ini tidak perlu dikirim lagi karena sudah disimpan di variabel hitungan yang berada di luar fungsi. Dengan perubahan ini, counter akhirnya bekerja sebagaimana mestinya, yaitu menyimpan dan memperbarui nilai secara berkelanjutan setiap kali digunakan.
