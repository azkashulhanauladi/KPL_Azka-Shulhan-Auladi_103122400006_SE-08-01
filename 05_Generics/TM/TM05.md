**Deskripsi Program**

Program ini dibuat untuk menjalankan konsep FizzBuzz, yaitu mengubah angka berdasarkan aturan tertentu.

Ada dua fungsi utama:

zzzzOrNum dipakai untuk mengecek satu angka. Kalau habis dibagi 3 jadi "Fizz", dibagi 5 jadi "Buzz", dan kalau keduanya jadi "FizzBuzz". Kalau tidak, tetap angka biasa. Fungsi ini juga ngecek supaya input harus berupa angka.

fizzBuzz dipakai untuk memproses array angka. Setiap elemen di array akan dicek pakai zzzzOrNum, lalu hasilnya dikumpulkan jadi array baru. Fungsi ini juga memastikan input harus berupa array.
