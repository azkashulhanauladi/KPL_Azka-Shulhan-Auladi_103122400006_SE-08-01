## Deskripsi Program
Fungsi toNumberArray berfungsi untuk mengubah berbagai jenis input menjadi array angka. Kalau inputnya sudah berupa array, fungsi langsung menggunakannya. Kalau inputnya masih berbentuk string, fungsi akan memecahnya menjadi beberapa bagian berdasarkan tanda koma. Setelah itu, setiap data dibersihkan dari spasi di awal atau akhir menggunakan trim(), lalu dicoba diubah menjadi angka dengan parseFloat(). Nilai yang tidak bisa dikonversi menjadi angka akan dihapus melalui proses filter(), sehingga array yang dihasilkan hanya berisi angka-angka yang valid dan siap digunakan.
